# Anti-AI-Qiang (抗 AI 腔)

`anti-ai-qiang` 是一个用于消除文本中 AI 痕迹和陈词滥调的提示词技能（Skill）。它融合了以下两个优秀的项目：
1. **[humanizer](https://github.com/blader/humanizer)**（22.4k 🌟）：基于维基百科 AI 清理项目（WikiProject AI Cleanup）详尽的“AI 写作迹象”指南。
2. **[stop-slop](https://github.com/hardikpandya/stop-slop)**（8.8k 🌟）：基于 Hardik Pandya 开发的 “Stop Slop” 提示词指南。

本项目通过多文件拆分和精细的融合，移除了无意义的客套垫话、大厂黑话、戏剧化断句、虚假施动者等 39 条 AI 特征模式，提供了灵活的声音微调方案、严格的评分系统与快速检查表，让大语言模型重新用有温度、长短错落、直截了当的真实人类声音进行写作。

## 核心功能与亮点

### 1. 39 条核心规则
我们将原项目的规则进行了深度融合与去重，划分为以下五大模式：
- **内容模式（规则 #1-#6）**：纠正对重要性/知名度的过度渲染，消除肤浅的“通过……展现”套路和模板化“挑战与前景”结尾。
- **语言与语法模式（规则 #7-#13, #31-#36）**：消除 AI 高频词、大厂黑话，纠正虚假施动者、被动语态、戏剧化断句、倒装句首词等。
- **格式与排版样式（规则 #14-#19）**：硬性禁用破折号（—），纠正滥用加粗、表情符号（Emojis）和混乱的引号。
- **沟通与废话模式（规则 #20-#22, #37-#39）**：剔除聊天助手客套话、截止日期免责，避免说教和刻意包装的“金句”。
- **填充与对冲（规则 #23-#30）**：删减冗余填充词，消除过度对冲（和稀泥）与伪造的权威感，规范文档注释的变化叙事。

### 2. 声音微调（Voice Tuning）
支持通过用户提供的写作样本来分析并模仿用户的个人文风（句子长度、词汇水平、标点习惯等），实现千人千面的去 AI 腔，而不仅仅是千篇一律地抹平文笔。

### 3. 个性与灵魂（Personality & Soul）
仅在有需要的语境（随笔、个人博客）中注入人性温度：长短句交错、表明鲜明立场、容许随性偏离，消除冰冷、旁观者的上帝视角。

### 4. 严格的评分系统 (Scoring System)

在交付改写文本前，大语言模型将对照 **快速检查表** 排除副词、破折号和无生命施动者，并对以下五个维度进行 1 到 10 分的评分：

| 评估维度 | 评估问题 |
|-----------|----------|
| **直截了当 (Directness)** | 是直接陈述事实，还是像在做预告说明？是否有无意义开场？ |
| **节奏变化 (Rhythm)** | 句子长短有致，还是像节拍器一样千篇一律？是否排除了三项式排比与多余破折号？ |
| **信任读者 (Trust)** | 是否尊重读者的智商（避免过度解释、恩准安慰、猜测填充与说教）？ |
| **真实感 (Authenticity)** | 听起来像真人写的吗？有具体的细节、情绪或第一人称立场吗？ |
| **信息密度 (Density)** | 是否有任何可以删减的虚词、大厂黑话或空泛的未来展望？ |

**总分低于 35/50 分：必须重新修改，直至达标。**
